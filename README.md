**Number Guessing Game — Project Description

📌 Project Title
Number Guessing Game — A Console-Based Java Application

📝 Overview
The Number Guessing Game is an interactive console-based application developed in Java. The program randomly generates a secret number within a defined range (1–100) and challenges the user to guess it within a limited number of attempts. The game provides real-time feedback after each guess and tracks the player's performance across multiple rounds.

🎯 Objectives

To apply core Java programming concepts such as loops, conditionals, and user input handling
To build an interactive and user-friendly command-line game
To implement game logic including scoring, attempt limits, and replay functionality


⚙️ Features

🔢 Random Number Generation within a specified range (1 to 100)
💬 Real-time Feedback — tells the user if their guess is too high, too low, or correct
🚫 Attempt Limit — maximum of 7 attempts per round
🔄 Multiple Rounds — player can choose to play again after each round
🏆 Score Tracking — displays rounds won, rounds played, and win rate at the end
✅ Input Validation — handles invalid entries gracefully
